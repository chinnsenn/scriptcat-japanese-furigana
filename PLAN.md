# ScriptCat 日语注音 1.0 计划

目标：交付一个可持续维护动态网页、可逆且隐私边界清晰的日语汉字注音用户脚本。

技术栈：JavaScript Userscript + ScriptCat GM API + Yahoo! JAPAN ルビ振り API + esbuild + Node Test Runner；沿用现有深模块结构，保持安装产物无运行时依赖。

## Sprint 01：精准采集与安全标注

- 选中文本、正文区域、整页三种范围均可直接标注。
- 导航、表单、代码、隐藏内容与已有 ruby 自动排除。
- 跨内联节点词语可生成一个连续 ruby，同时保留元素结构、样式和事件。
- 撤销后页面 DOM 与原始文本完整恢复。

## Sprint 02：持续会话与稳定分析

- SPA 导航、无限滚动、文本改写和局部重渲染持续进入当前注音会话。
- 按钮显示 `处理中 n/total`，处理期间再次点击可取消。
- 429 退避与瞬时错误重试共享统一调度；单片段失败保留其余结果。
- 失败片段可重试，统计面板展示进度、等待、失败与耗时。

## Sprint 03：站点隐私控制

- 站点 origin 独立保存正文发送许可与自动标注白名单。
- 菜单可选择默认范围、撤销许可、清理当前站点缓存和管理 Client ID。
- 用户可在发送前查看本次实际发送范围与文本摘要。
- 自动标注仅在白名单站点启用。

## Sprint 04：自动发布与 1.0 验收

- GitHub 标签发布自动构建并校验单文件产物。
- Greasy Fork Webhook 由 GitHub Actions 安全触发，密钥只存在于仓库 Secrets。
- README 记录安装、发布配置、隐私边界与完整验收命令。
- 逐项审计 1.0 产品目标并留下可复现证据。
